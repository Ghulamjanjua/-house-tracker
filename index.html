Setup Instructions
1

Create a new Google Sheet

Go to sheets.google.com and create a blank spreadsheet. Name it House Build Tracker.

2

Open Apps Script editor

Click Extensions → Apps Script. Delete all existing code. Paste the entire Code.gs file. Press Ctrl+S.

3

Run setup once

Select setupSheets from the function dropdown. Click Run ▶. Accept all permissions.

4

Deploy as Web App

Deploy → New deployment → Web app. Execute as: Me. Access: Anyone. Deploy → Authorize → Copy the URL ending in /exec.
