Google Sheets Connection
Setup Instructions — Follow in Order
1

Create a new Google Sheet

Go to sheets.google.com and create a blank spreadsheet. Name it House Build Tracker. Copy the Sheet ID from the URL — the long string between /d/ and /edit. You won't need to paste it anywhere; the Apps Script handles the connection.

2

Open Apps Script editor

In your Sheet, click Extensions → Apps Script. Delete all existing code. Paste the entire contents of the Code.gs file you downloaded. Press Ctrl+S to save.

3

Run the setup function once

In the Apps Script editor, select setupSheets from the function dropdown (top toolbar) and click Run ▶. Accept any permissions Google asks for. This creates the Invoices, Budget, and Audit tabs in your Sheet automatically.

4

Deploy as Web App

Click Deploy → New deployment. Click the gear ⚙ icon next to "Type" and select Web app. Set Execute as: Me. Set Who has access: Anyone. Click Deploy. Click Authorize access and follow the prompts. Copy the Web app URL — it ends in /exec.

5

Paste URL below and connect

Paste the Web app URL into the field below and click Connect. Your data will sync to Google Sheets from that point forward.
